= Text Fragments

A text fragment is a small section of text that is stored in an AsciiDoc file. Text fragments contain content that is resused in multiple modules or assemblies, for example:

* A step or series of steps in a procedure
* A disclaimer, for example, for technology preview or beta releases

NOTE: A text fragment is not a module.

.Procedure
. Create the text fragment AsciiDoc file.
+
NOTE: Consider storing fragment files in a separate fragments folder.
. Add an `include::` statement to the file that you want to add the fragment to, for example:
+
[source]
----
include::fragments/beta-note.adoc
----
